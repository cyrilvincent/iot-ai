TP 12 : BLE
Installer le package aioble sur l'ESP32
Exécuter le script ble_server.py /!\ Les UUID doivent être unique dans la salle
Installer nRF Connect sur votre téléphone
Démarrer nRF Connect, votre ESp32 doit apparaitre, connectez vous
Chercher Unknown Service avec l'UUID de _BLE_SENSOR_CHAR_UUID
Chercher la caractéristique qui est NOTIFY, READ : vous être sur sensor_task
Cliquer sur la flêche qui descend, la valeur est dans le champ value
Chercher la caractéristique qui est NOTIFY, READ, WRITE : vous être sur wait_for_write
Cliquer sur la fleche qui monte
Dans le menu déroulant choisir le type TEXT (UTF-8)
Saisir 1 dans New Value : la led s'allume
Saisir 0 : la led s'eteint
Remplacer le service sensor_task par une mesure réelle de température
Tester sur téléphone
Bonus : Créer 2 autres services avec humidity et pressure
Bonus : Créer un service qui affiche un message sur LCD ou OLED
Bonus : Créer le client Python (ble_client.py)
Bonus : Tester chez vos voisins

TP 15 : MQTT
Copier lib/broker.emqx.io-ca.crt sur l'ESP32
Exécuter le code mqtt_emqx_client.py 
Sur https://mqttx.app/downloads télécharger et installer le client MQTTX Windows Universal
Ou utiliser l'application téléphonique MyMQTT et activer SSL et v3
Créer une nouvelle connexion : name=formation et cliquer sur connect
Cliquer sur new subscription et topic=raspberry/mqtt
Vous verrez apparaitre vos message et vous pouvez poster un message en JSON que vous verrez apparaitre sur l'ESP32
